# Agent Relay GitHub Update Usage

## Good Moments To Trigger

- after a benchmark or training run
- before switching to a new session
- before handing work to another teammate or agent
- after deciding a branch is ruled out
- when GitHub relay state fell behind the actual repository state

## Example Prompts

- `Use $agent-relay-github-update to refresh the active relay issue and post a clean handoff comment.`
- `Use $agent-relay-github-update to record this benchmark run as the current trusted best in GitHub.`
- `Use $agent-relay-github-update to leave a ready-to-paste handoff if direct GitHub write access is unavailable.`
