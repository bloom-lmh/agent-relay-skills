# Agent Relay Local Handoff Usage

## Good Triggers

- resume a repository in a new session
- recover context from docs instead of chat history
- check the trusted best checkpoint and next step
- summarize current relay state before deciding what to do next

## Example Prompts

- `Use $agent-relay-local-handoff to resume this repository and tell me the current best checkpoint and next step.`
- `Use $agent-relay-local-handoff to recover context from this repo instead of chat history.`
- `Use $agent-relay-local-handoff to read AGENTS.md and the relay docs, then summarize the current mainline.`
