# Agent Relay GitHub Handoff Usage

## Good Triggers

- resume a GitHub-tracked task in a new session
- summarize the active relay issue
- recover trusted best and next action from issue comments
- continue work without relying on previous chat history

## Example Prompts

- `Use $agent-relay-github-handoff to resume this repository from the active GitHub relay issue and summarize the next step.`
- `Use $agent-relay-github-handoff to recover the trusted best checkpoint and blockers from GitHub comments.`
